# prac_bar

Es una aplicación que simula los pedidos en un bar.
Se utilizo:
    -Framework: Flutter (v3.x+)
    -Lenguaje: Dart
    -Arquitectura: MVVM (Model-View-ViewModel).
Las características principales de la aplicación son: Creacion de pedidos,visualizacion de pedidos, en la creación de pedidos permite ver un resumen del mismo sin realizarlo.
